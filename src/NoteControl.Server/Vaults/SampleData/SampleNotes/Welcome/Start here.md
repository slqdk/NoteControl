# Welcome to NoteControl

This is a sample note. It was created automatically when you clicked **Install Sample Data** on this vault. Open the other notes in the **Welcome**, **Examples**, and **Daily journal** folders to see what NoteControl can do.

<div class="nc-callout nc-callout-tip" data-variant="tip">

**Tour the editor.** Press `/` anywhere in this note (or any note) to open the slash menu. From there you can insert headings, lists, code blocks, callouts, tables, images, and more — without learning any markdown syntax.

</div>

## What's in this vault

The sample data installer created three folders:

- **Welcome** — this note plus a slash-menu reference.
- **Examples** — code blocks (incl. Structured Text), tables, images, links.
- **Daily journal** — what a daily note looks like.

Delete any of these any time; the installer never re-creates files that already exist, so you can safely curate.

## Where the actual files live

Open Windows Explorer and navigate to your vault folder. Every note you see in NoteControl is a `.md` file you can open in Notepad, Obsidian, Logseq, or any editor that understands markdown. Folders are folders. Attachments live in a sibling `{note-name}.assets/` folder next to each note.

If NoteControl ever stops working, your data is still there in plain text. Backup = copy the folder. Restore = paste it back.

## Next steps

1. Click around — try the slash menu, paste a screenshot into a note, drag-rename a note, drop a tag on something.
2. Visit the Startpage (🏠 icon at the top of the tree) to set up RSS feeds and task areas for this vault.
3. Try Daily Notes (📅 icon at the top of the tree) to start journaling.

Have fun. None of this is magic — it's just files.
